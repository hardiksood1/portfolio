Real-Time Person Detection, Tracking, and Face Recognition

Advanced surveillance system with ROI alerts

Project Overview

Developed a robust surveillance system that detects and tracks persons in a video feed, recognizes known faces, logs entry and exit within polygonal ROIs, and sends WhatsApp alerts when persons enter sensitive regions. The system streams processed video live over RTSP for real-time monitoring.

Key Features

  • Real-time person detection using YOLOv8 and face recognition
  • Multi-object tracking with DeepSORT for consistent ID assignment
  • Identification of known vs unknown individuals
  • Custom polygonal ROI drawing using OpenCV
  • Entry/exit tracking with time spent calculation
  • Automated WhatsApp alerts with person ID, name, timestamp and snapshot
  • Docker-based RTSP server for flexible deployment
  • Real-time stream monitoring via VLC or any RTSP-compatible client

LLM-based Yield Prediction using OpenLLaMA & LoRA Fine-Tuning

NLP for regression task predicting crop yield

Project Overview

Fine-tuned a 7B OpenLLaMA model using LoRA adapters for a regression task predicting crop yield from structured tabular data. Engineered prompt templates to convert regression into a text generation task, achieving scalable deployment-ready output with predictions stored in Excel.

Key Features

  • Fine-tuned OpenLLaMA 7B model using LoRA adapters
  • 4-bit quantization and PEFT for memory-efficient training
  • Engineered prompt templates from Excel data
  • Converted regression task to text generation
  • Evaluated using RMSE, MAE, and R² metrics
  • Implemented JSONL dataset pipeline using Hugging Face Datasets
  • Scalable deployment with predictions stored in Excel
